## Language Translation Tool

## Project Overview
This project is a simple Language Translation Tool developed using Python. It translates text from one language to another using the Deep Translator library. The application provides a user-friendly interface using Gradio for easy language translation.

## Features
- Translate text between multiple languages
- Simple and interactive user interface
- Fast and accurate translations
- Supports various languages

## Technologies Used
- Python
- Google Colab
- Deep Translator
- Gradio

## How to Run
1. Install the required libraries:
   ```
   pip install deep-translator gradio
   ```
2. Run in the Python script or Google   Colab notebook.
3. Enter the input text.
4. Select the source and target languages.
5. Click **Translate** to view the translated text.
